# Mi Frontend React - CRUD de Productos

## Descripción
Este proyecto es una aplicación frontend desarrollada en React que se conecta a un backend mediante solicitudes HTTP (API REST). Cuenta con la funcionalidad completa de un CRUD que permite listar datos (GET), registrar nueva información a través de un formulario (POST), modificar registros existentes (PUT) y remover elementos del sistema con confirmación previa (DELETE). El proyecto está modularizado y optimizado mediante componentes separados para mejorar el rendimiento.

## Tecnologías utilizadas
* **React.js** (Librería principal de JavaScript)
* **Vite** (Herramienta de construcción y entorno de desarrollo rápido)
* **JavaScript (ES6+)**
* **CSS e inline styles** para el diseño de la interfaz
* **Fetch API** para el consumo y conectividad con servicios web externos

## Instrucciones de instalación
Para instalar y configurar el proyecto de manera local, ejecute los siguientes comandos en su terminal:

1. Clonar el repositorio remoto:
   ```bash
   git clone https://github.com
   ```
2. Navegar a la carpeta del proyecto:
   ```bash
   cd mi-frontend-react
   ```
3. Instalar todas las dependencias necesarias:
   ```bash
   npm install
   ```

## Forma de ejecución
Una vez instaladas las dependencias, inicie el servidor de desarrollo local ejecutando:

```bash
npm run dev
```
Abra su navegador web e ingrese a la dirección local indicada por la consola (usualmente `http://localhost:5173`).
